HTML Lists

1. Ordered List

An ordered list (OL) displays items in a numbered sequence.

  1. First item
  2. Second item
  3. Third item

2. Unordered List

An unordered list (UL) displays items with bullet points.

  • Item 1
  • Item 2
  • Item 3

3. Nested List

A nested list contains a list within another list.

  • Fruits
    • Apple
    • Banana
    • Cherry
  • Vegetables
    • Carrot
    • Spinach
    • Broccoli

4. Ordered List Inside Unordered List

An ordered list inside an unordered list.

  • Shopping List:
    1. Milk
    2. Eggs
    3. Bread
  • To-Do List:
    1. Write report
    2. Attend meeting
    3. Buy groceries

5. Definition List

A definition list is used to define terms and their descriptions.

HTML
HyperText Markup Language - the standard markup language for web pages.
CSS
Cascading Style Sheets - used to style the layout of web pages.
JavaScript
A programming language used to create interactive effects in web browsers.
